# RXF Studio Pro

**RXF Studio Pro** is a professional-grade application for high-performance media processing, real-time visual effects, and advanced compositing. It combines a powerful GPU-accelerated rendering engine with a flexible node-based workflow, delivering desktop-class performance exclusively for Windows.

---

## Key Features

- **Non-destructive workflow** – all changes are procedural, original files remain untouched
- **Node-based graph** – visually build complex processing pipelines without coding
- **Real-time preview** – low latency playback even with heavy effect chains
- **Broad format support** – high-bit-depth sequences, alpha channel, all major professional codecs
- **Built-in tools** – color grading with LUT, particle systems, transitions, keying, masking, procedural generation
- **Batch processing** – automate rendering of multiple projects in sequence
- **GPU acceleration** – full support for NVIDIA CUDA and AMD OpenCL

## System Requirements

- **Operating System:** Windows 10 (version 22H2 or later) or Windows 11
- **VRAM:** 4 GB minimum, 8 GB or more recommended
- **RAM:** 16 GB for HD workflows, 32 GB for 4K, 64 GB for 8K
- **Storage:** 2 GB for installation, SSD recommended for cache and temp files
- **Display:** 1920x1080 or higher, support for 10-bit color recommended

---

## Quick Start

1. **Create a new project** – File > New Project
2. **Import media** – Drag and drop files into the node graph
3. **Build your pipeline** – Connect nodes (input > filter > output)
4. **Preview** – Click the Play button to view real-time results
5. **Export** – Configure output format and render to disk

## Troubleshooting

**Q: The application crashes on startup**  
A: Update your GPU and CPU drivers to the latest version. Ensure Windows is fully updated.

**Q: Preview is lagging**  
A: Reduce preview resolution using the quality dropdown. Close other GPU-intensive applications.

**Q: Out of memory error**  
A: Increase system RAM or reduce timeline resolution. Enable disk caching in Settings.
